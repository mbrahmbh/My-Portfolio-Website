01

EEBOT - Robot Intelligence

Programmed an Intelligent autonomous microprocessor robot in a team of 2 with obstacle detection, dead-end recognition,and route optimization functionalities. Demonstrated a working prototype of robot's ability to use basic intelligence capable of solving mazes by navigating to a designated point and returning back to the starting position without human intervention. The EEBOT's HCS12 microprocessor was programmed in assembly using CodeWarrior IDE.

03

General Purpose Processor Design

Designed and built General Purpose Processor, GPP, using Altera Quartus II and VHDL, for my Digital Systems course. The GPP consisted of two storage units made with latches, a control unit made using a finite state machine and 4x16 Decoder which feed input to an ALU which then displays output on seven segment display.All components were programmed using VHDL, final design was then implemented on the FPGA board
